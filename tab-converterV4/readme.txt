このツールは、4弦ベースのTAB譜を5弦ベース用に最適化されたTAB譜へ変換するWebアプリケーションです。 

🌟 使い方（とっても簡単です！）
テンプレート取得: まずは「テンプレート取得」ボタンを押し、TAB譜の正しい書き方（フォーマット）を確認します。 

TAB譜の編集: 取得したテンプレートを元に、ご自身のTAB譜を追記・編集してください。 

入力欄に貼り付け: 編集したTAB譜をコピーし、入力欄に貼り付けます。 

フォーマットチェック: 「フォーマットチェック」ボタンを押し、TAB譜の形式に間違いがないかを確認します。 

5弦用に変換: チェックで問題がなければ、「5弦TABに変換」ボタンを押すと、変換された結果が出力エリアに表示されます。 

ダウンロード: 最後に「変換結果をダウンロード」ボタンを押すと、結果を.txtファイルとして保存できます。 

📌 将来のアップデートについて
現在は.txt形式でのダウンロードのみ対応していますが、将来的にはPDF形式など、他のフォーマットでの保存機能も追加する予定です。

This tool is a simple web application that converts 4-string bass TABs into optimized 5-string bass TABs. 

🌟 How to Use (It's super easy!)

Get Template: First, click the “Get Template” button to see the standard TAB format. 


Edit Your TAB: Add or edit your own musical score based on the template. 


Paste into Input Area: Copy and paste your completed TAB into the input field. 


Check Format: Use the “Check Format” button to verify that your TAB is written in a valid format. 

Convert to 5-String: If the format is correct, click the “Convert to 5-string TAB” button. The result will be displayed in the output area below. 


Download: Finally, use the “Download” button to save the result as a .txt file. 

📌 Future Updates
Currently, downloading is supported in .txt format only, but we plan to add other formats like .pdf in future updates!
